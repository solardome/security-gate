# Code of Conduct

## Our Standards

We are committed to a respectful, inclusive, and collaborative community.

Examples of positive behavior:

- Being respectful in technical discussions
- Giving constructive, actionable feedback
- Focusing critique on ideas and code, not people

Examples of unacceptable behavior:

- Harassment, discrimination, or personal attacks
- Trolling, insulting, or inflammatory comments
- Publishing private information without consent

## Enforcement

Maintainers are responsible for clarifying and enforcing this code of conduct.

If behavior violates these standards, maintainers may remove content, restrict
participation, or take other appropriate action.

## Reporting

Report incidents privately to the maintainers.

## Scope

This code of conduct applies in project spaces, including issues, pull requests,
commit discussions, and any official communication channels.
